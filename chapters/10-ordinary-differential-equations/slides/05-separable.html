10.6.1   Variables Separable Method

Spot the Clue

Every term is a product of a pure-$x$ piece and a pure-$y$ piece. You can divide to isolate them.

$$\underbrace{f_1(x)}_{\text{only }x}\!\cdot\!\underbrace{g_1(y)}_{\text{only }y}\,dx \;+\; \underbrace{f_2(x)}_{\text{only }x}\!\cdot\!\underbrace{g_2(y)}_{\text{only }y}\,dy = 0$$
↓ divide by $f_2(x)\,g_1(y)$
$$\overbrace{\frac{f_1(x)}{f_2(x)}\,dx}^{\text{pure } x} = \overbrace{-\frac{g_2(y)}{g_1(y)}\,dy}^{\text{pure } y}$$
↓ integrate both sides
$$\boxed{\int \frac{f_1(x)}{f_2(x)}\,dx = -\int \frac{g_2(y)}{g_1(y)}\,dy + c}$$

The Key Test: Can You Factor?

$$\underbrace{(1+x^2)}_{\text{only }x}\frac{dy}{dx} = \underbrace{(1+y^2)}_{\text{only }y}$$ Each side is a single-variable function. Separable!
$$\underbrace{x^2}_{\text{only }x}\!\cdot\!\underbrace{y}_{\text{only }y}\,dx + \underbrace{(1+x^3)}_{\text{only }x}\,dy = 0$$ Each term = f(x) × g(y). Separable!

Ex: $(1+x^2)\dfrac{dy}{dx} = 1+y^2$

$$\frac{dy}{1+y^2}=\frac{dx}{1+x^2}$$
$$\tan^{-1}y = \tan^{-1}x + c$$
$$\boxed{y-x = a(1+xy)}$$

Ex: $(1\!+\!x^3)dy - x^2 y\,dx = 0$

$$\frac{dy}{y}=\frac{x^2}{1+x^3}\,dx$$
$$\log y = \tfrac{1}{3}\log(1\!+\!x^3)+\log c$$
$$\boxed{y^3 = c(1+x^3)}$$

$y(1)=2 \Rightarrow c=4$

Can't Separate When...

$x$ and $y$ are added inside a term — no way to factor into $f(x)\cdot g(y)$.

$$\frac{dy}{dx} = \underbrace{x + y}_{\text{sum, not product}}$$ Can't write as $h(x)\cdot k(y)$. → Use Linear ($y'+(-1)y = x$)
$$\frac{dy}{dx} = \underbrace{x^2 + xy + y^2}_{\text{mixed terms}}$$ $xy$ term couples $x,y$. → Try Homogeneous ($y=vx$)
