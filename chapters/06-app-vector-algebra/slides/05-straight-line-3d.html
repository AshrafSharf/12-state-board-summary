6.5   Equation of a Straight Line in 3D

Core Idea

A straight line in 3D is determined by either (a) a point and a direction, or (b) two points. The equation can be expressed in parametric, non-parametric (vector), or Cartesian form.

$$\boxed{\text{Parametric: } \vec{r} = \vec{a} + t\vec{b}, \; t \in \mathbb{R}}$$
$$\boxed{\text{Cartesian: } \frac{x - x_1}{b_1} = \frac{y - y_1}{b_2} = \frac{z - z_1}{b_3}}$$

Visual: Line Through Point $A$ in Direction $\vec{b}$

The line passes through $A$ (position vector $\vec{a}$) and extends in direction $\vec{b}$. Parameter $t$ traces the whole line.

Other Forms

$$\text{Non-parametric: } (\vec{r} - \vec{a}) \times \vec{b} = \vec{0}$$
$$\text{Through two points: } \vec{r} = \vec{a} + t(\vec{b} - \vec{a}), \; t \in \mathbb{R}$$
$$\text{Cartesian (two points): } \frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1} = \frac{z - z_1}{z_2 - z_1}$$
$$\frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1} = \frac{z - z_1}{z_2 - z_1}$$

Spot the Clue

  • "Line passing through ... and parallel to ..." → point-direction form
  • "Line through two points" → direction = difference of position vectors
  • "Direction ratios" or "direction cosines" tell you the $\vec{b}$ vector
  • "Parallel to a given line" → use the SAME direction vector, not the point on that line

Ex: Line Through a Point, Parallel to a Vector (Ex 6.24)

Line passes through $(1,2,-3)$, parallel to $4\hat{i}+5\hat{j}-7\hat{k}$.

Parametric: $\vec{r} = (\hat{i}+2\hat{j}-3\hat{k}) + t(4\hat{i}+5\hat{j}-7\hat{k})$

Non-parametric: $(\vec{r} - (\hat{i}+2\hat{j}-3\hat{k})) \times (4\hat{i}+5\hat{j}-7\hat{k}) = \vec{0}$

$$\boxed{\frac{x-1}{4} = \frac{y-2}{5} = \frac{z+3}{-7}}$$

Watch Out

$$\frac{x-1}{0} \text{ is undefined}$$ Don't panic at zero denominators
$$b_3=0 \Rightarrow z - z_1 = 0$$ Zero denominator means that variable is constant

Trap: When reading direction ratios from $\frac{-x-2}{4} = \frac{y+3}{-2} = \frac{2z-6}{3}$, rewrite as $\frac{x+2}{-4} = \frac{y+3}{-2} = \frac{z-3}{3/2}$ first.
