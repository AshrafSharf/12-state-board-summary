6.8   Angle Between Planes, Line-Plane Angle & Distance

Core Idea

The angle between two planes equals the angle between their normals. The angle between a line and a plane is the complement of the angle between the line's direction and the plane's normal. Distance formulas give perpendicular distances from points to planes and between parallel planes.

$$\boxed{\text{Angle between planes: } \theta = \cos^{-1}\!\left(\frac{|\vec{n}_1 \cdot \vec{n}_2|}{|\vec{n}_1||\vec{n}_2|}\right)}$$
$$\boxed{\theta_{\text{planes}} = \cos^{-1}\!\left(\frac{|\vec{n}_1 \!\cdot\! \vec{n}_2|}{|\vec{n}_1||\vec{n}_2|}\right)}$$
$$\boxed{\text{Line-plane angle: } \theta = \sin^{-1}\!\left(\frac{|\vec{b} \cdot \vec{n}|}{|\vec{b}||\vec{n}|}\right)}$$
$$\boxed{\theta_{\text{line-plane}} = \sin^{-1}\!\left(\frac{|\vec{b} \!\cdot\! \vec{n}|}{|\vec{b}||\vec{n}|}\right)}$$

Distance Formulas

$$\boxed{\text{Point to plane: } \delta = \frac{|ax_1+by_1+cz_1+d|}{\sqrt{a^2+b^2+c^2}}}$$
$$\boxed{\delta = \frac{|ax_1\!+\!by_1\!+\!cz_1\!+\!d|}{\sqrt{a^2\!+\!b^2\!+\!c^2}}}$$
$$\boxed{\text{Between parallel planes: } \delta = \frac{|d_1 - d_2|}{\sqrt{a^2+b^2+c^2}}}$$
$$\text{Vector form: } \delta = \frac{|\vec{u} \cdot \vec{n} - p|}{|\vec{n}|}$$

Parallel & Perpendicular Conditions

$$\text{Planes } \parallel \iff \vec{n}_1 = \lambda\vec{n}_2$$Proportional normals
$$\text{Planes } \perp \iff \vec{n}_1 \cdot \vec{n}_2 = 0$$Orthogonal normals
$$\text{Line } \perp \text{ plane} \iff \vec{b} = \lambda\vec{n}$$Direction parallel to normal
$$\text{Line } \parallel \text{ plane} \iff \vec{b} \cdot \vec{n} = 0$$Direction perpendicular to normal

Spot the Clue

  • "Angle between the planes" → extract normal vectors, use $\cos^{-1}$
  • "Angle between line and plane" → use $\sin^{-1}$ (NOT $\cos^{-1}$!)
  • "Distance from point to plane" → direct substitution
  • "Parallel planes" → same normal ratios; distance uses $|d_1 - d_2|$

Ex: Point-to-Plane Distance (Ex 6.49)

Point $(2,5,-3)$, plane $\vec{r} \cdot (6\hat{i}-3\hat{j}+2\hat{k}) = 5$

Step 1: $\vec{u} \cdot \vec{n} = 12 - 15 - 6 = -9$

Step 2: $|\vec{n}| = \sqrt{36+9+4} = 7$

$$\boxed{\delta = \frac{|-9 - 5|}{7} = \frac{14}{7} = 2 \text{ units}}$$

Watch Out

$$\text{Line-plane angle } = \cos^{-1}\!\left(\frac{|\vec{b} \cdot \vec{n}|}{|\vec{b}||\vec{n}|}\right)$$ This gives the complement!
$$\text{Line-plane angle } = \sin^{-1}\!\left(\frac{|\vec{b} \cdot \vec{n}|}{|\vec{b}||\vec{n}|}\right)$$ Use $\sin^{-1}$ for line-plane

Trap: For parallel planes $ax+by+cz+d_1=0$ and $2ax+2by+2cz+d_2=0$, you must first scale so the $a,b,c$ coefficients are identical before computing $|d_1-d_2|$.
