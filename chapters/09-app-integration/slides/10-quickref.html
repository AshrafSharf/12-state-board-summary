Indefinite Integral Toolkit — Pattern Recognition

U-Substitution

Clue: $f(g(x))\cdot g'(x)$ — derivative of inner function is present

Action: Let $u=g(x)$, replace, integrate, sub back

Integration by Parts (LIATE)

Clue: Product of unrelated functions, u-sub fails

Choose $u$: L ≻ I ≻ A ≻ T ≻ E

Partial Fractions

Clue: $\frac{P(x)}{Q(x)}$ where $Q(x)$ factors into linear/quadratic terms

Decompose: $\frac{1}{(x-a)(x-b)} = \frac{A}{x-a}+\frac{B}{x-b}$

Completing the Square

Clue: $\frac{1}{ax^2+bx+c}$ where quadratic doesn't factor nicely

Rewrite: $(x-h)^2 \pm k^2$ → use $\arctan$ or log forms

Trig Substitution Table

Clue: Integrand has $\sqrt{a^2-x^2}$, $\sqrt{a^2+x^2}$, or $\sqrt{x^2-a^2}$ and u-sub can't remove the root.

If you seeSubstituteIdentity used
$\sqrt{a^2-x^2}$$x = a\sin\theta$,   $dx = a\cos\theta\,d\theta$$1-\sin^2\!\theta = \cos^2\!\theta$
$\sqrt{a^2+x^2}$$x = a\tan\theta$,   $dx = a\sec^2\!\theta\,d\theta$$1+\tan^2\!\theta = \sec^2\!\theta$
$\sqrt{x^2-a^2}$$x = a\sec\theta$,   $dx = a\sec\theta\tan\theta\,d\theta$$\sec^2\!\theta-1 = \tan^2\!\theta$
$$\int\!\sqrt{4-x^2}\,dx \;\xrightarrow{x=2\sin\theta}\; \int 4\cos^2\!\theta\,d\theta \;\longrightarrow\; 2\theta+\sin 2\theta + C$$ Square root eliminated! Then use half-angle identity. Finally sub back $\theta = \sin^{-1}(x/2)$.

Partial Fractions: The Setup

Factor the denominator, then decompose. Each fraction integrates to a $\ln$ or power.

Distinct linear factors

$$\frac{1}{(x\!-\!2)(x\!-\!3)} = \frac{A}{x-2}+\frac{B}{x-3}$$

Solve: $A=-1, B=1$ → $\int = \ln\left|\frac{x-3}{x-2}\right|+C$

Repeated linear factor

$$\frac{1}{(x\!-\!1)^2(x\!+\!2)} = \frac{A}{x-1}+\frac{B}{(x-1)^2}+\frac{C}{x+2}$$

Repeated factors need one fraction per power.

Completing the Square: Non-Factorable Quadratics

If $ax^2+bx+c$ doesn't factor, rewrite as $(x-h)^2\pm k^2$ and use standard forms:

$$\int\!\frac{dx}{x^2-5x+6} \;\xrightarrow{\text{factor}}\; \int\!\frac{dx}{(x-2)(x-3)}$$ Factors → partial fractions
$$\int\!\frac{dx}{x^2-5x+7} \;\xrightarrow{\text{complete}}\; \int\!\frac{dx}{(x-\frac{5}{2})^2+\frac{3}{4}}$$ Doesn't factor → $\arctan$ form

All Key Formulas at a Glance

$$\underbrace{\int_a^b f(x)\,dx = F(b)-F(a)}_{\text{Second FTC}} \qquad \underbrace{\int_a^b f(x)\,dx = \int_a^b f(a+b-x)\,dx}_{\text{P6: Mirror}}$$
$$\underbrace{\int_{-a}^{a}f(x)\,dx = \begin{cases}2\int_0^a f & \text{even}\\0 & \text{odd}\end{cases}}_{\text{P8/P9}} \qquad \underbrace{\int_0^a x\,f(x)\,dx = \frac{a}{2}\int_0^a f(x)\,dx}_{\text{P12, if } f(a-x)=f(x)}$$
$$\underbrace{\int_0^{\pi/2}\sin^n\!x\,dx = \int_0^{\pi/2}\cos^n\!x\,dx}_{\text{Wallis}} \qquad \underbrace{\Gamma(n)=\int_0^{\infty}e^{-x}x^{n-1}\,dx=(n\!-\!1)!}_{\text{Gamma}}$$
$$\underbrace{A = \int_a^b[y_U-y_L]\,dx}_{\text{Area between curves}} \qquad \underbrace{V = \pi\int_a^b y^2\,dx}_{\text{Volume about } x\text{-axis}}$$
