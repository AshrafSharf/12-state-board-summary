YES
U-Substitution

CHECK 5: Does it match special trig/inverse-trig structures?

You seeResult
$\dfrac{1}{a^2+x^2}$ $\dfrac{1}{a}\tan^{-1}\!\dfrac{x}{a}+C$
$\dfrac{1}{\sqrt{a^2-x^2}}$ $\sin^{-1}\!\dfrac{x}{a}+C$
$\dfrac{1}{a^2-x^2}$ $\dfrac{1}{2a}\ln\!\left|\dfrac{a+x}{a-x}\right|+C$

CHECK 6: Is it a product of two different types?

($x\cdot e^x$,   $x\cdot\sin x$,   $\ln x\cdot x$, etc.)

$$\text{Use Integration by Parts:}\quad \int u\,dv = uv - \int v\,du$$
$$\int u\,dv = uv - \int v\,du$$

Choose $u$ using LIATE: Log ≻ Inverse trig ≻ Algebraic ≻ Trig ≻ Exponential

$$\int x\,e^x\,dx$$ Product of $x$ (A) and $e^x$ (E) → IBP works
$$\int x\,e^{x^2}\,dx$$ Hidden substitution! $u=x^2$ is better

Secondary Detection — If Nothing Above Worked

  • Rational function? → Partial fractions / algebraic manipulation
  • Square root of quadratic? → Trig substitution ($x=a\sin\theta$, $a\tan\theta$, $a\sec\theta$)
  • Even/odd trig powers? → Use identities (half-angle, Pythagorean)
  • Completely unfamiliar? → Might be non-elementary (e.g., $\int e^{x^2}\,dx$)

Micro-Pattern Memory — 5 Derivatives That Solve Most Integrals

$$1)\;\frac{d}{dx}(\ln x) = \frac{1}{x}$$
$$2)\;\frac{d}{dx}(\tan^{-1} x) = \frac{1}{1+x^2}$$
$$3)\;\frac{d}{dx}(\sin x) = \cos x$$
$$4)\;\frac{d}{dx}(e^x) = e^x$$
$$5)\;\frac{d}{dx}(\ln|f(x)|) = \frac{f'(x)}{f(x)}$$

Common Traps

  • Looks like log? → Check numerator carefully — is it really $f'(x)$?
  • Looks like product? → Check for hidden substitution first before IBP
  • Looks like inverse trig? → Compare sign inside the square root / denominator
  • Power rule? → Check if exponent is $-1$ (power rule fails, log appears)
$$\int\frac{2x}{x^2+1}\,dx \to \ln(x^2+1)+C$$ Numerator IS derivative of denom
$$\int\frac{1}{x^2+1}\,dx \neq \ln(x^2+1)$$ No derivative on top! It's $\tan^{-1}\!x$

Master Flow — 6-Step Priority

  1. Simplify first — expand, factor, reduce
  2. Scan for $f'/f$ — derivative-over-function → $\ln$
  3. Scan for inner function — u-substitution
  4. Check standard trig forms — $\frac{1}{a^2+x^2}$, $\frac{1}{\sqrt{a^2-x^2}}$
  5. Try parts if product — use LIATE to choose $u$
  6. Only then go to advanced techniques (partial fractions, trig sub, etc.)

"Integration is not about memorizing 100 formulas. It is about recognizing 10 derivative patterns."
