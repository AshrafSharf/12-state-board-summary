9.1   Indefinite Integrals — The Building Blocks

Core Idea

If $\dfrac{d}{dx}F(x) = f(x)$, then $F(x)$ is an anti-derivative of $f(x)$.

$$\int f(x)\,dx = F(x) + C \quad \longleftarrow \text{ "+C" because } \frac{d}{dx}[F(x)+C] = f(x)$$

Linear property: $\displaystyle\int[\alpha f(x)+\beta g(x)]\,dx = \alpha\!\int\! f(x)\,dx + \beta\!\int\! g(x)\,dx$

Standard Integrals You Must Know

$f(x)$$\int f(x)\,dx$Pattern clue
$(ax+b)^n,\;n\neq-1$$\dfrac{1}{a}\cdot\dfrac{(ax+b)^{n+1}}{n+1}+C$Power rule — bump power by 1
$\dfrac{1}{ax+b}$$\dfrac{1}{a}\log|ax+b|+C$$n=-1$ case → log
$e^{ax}$$\dfrac{e^{ax}}{a}+C$Exponential stays exponential
$\sin(ax+b)$$-\dfrac{\cos(ax+b)}{a}+C$sin → −cos
$\cos(ax+b)$$\dfrac{\sin(ax+b)}{a}+C$cos → +sin
$\sec^2(ax+b)$$\dfrac{\tan(ax+b)}{a}+C$sec² → tan
$\dfrac{1}{a^2+x^2}$$\dfrac{1}{a}\tan^{-1}\!\left(\dfrac{x}{a}\right)+C$Sum of squares → arctan
$\dfrac{1}{\sqrt{a^2-x^2}}$$\sin^{-1}\!\left(\dfrac{x}{a}\right)+C$Root of diff of squares → arcsin
$\dfrac{1}{a^2-x^2}$$\dfrac{1}{2a}\log\left|\dfrac{a+x}{a-x}\right|+C$Partial fractions → log ratio

Generalization Tip: The Chain Factor $\frac{1}{a}$

Whenever the argument is $ax+b$ instead of just $x$, divide by $a$ (the coefficient of $x$). This comes from the chain rule in reverse.

$$\int\cos(3x+1)\,dx = \frac{\sin(3x+1)}{\color{#059669}{3}}+C$$ Divide by coefficient of $x$
$$\int\cos(x^2)\,dx \neq \frac{\sin(x^2)}{2x}+C$$ Argument is $x^2$ not $ax+b$. Need substitution!

When the $\frac{1}{a}$ Trick Fails

The $\frac{1}{a}$ shortcut works only when the inner function is linear ($ax+b$). If the argument is nonlinear ($x^2$, $\sqrt{x}$, $\sin x$, etc.), you must use substitution.

$$\int e^{x^2}\,dx$$ No closed form! Can't pull out "$\frac{1}{2x}$"
$$\int 2x\,e^{x^2}\,dx = e^{x^2}+C$$ The derivative of $x^2$ is sitting outside → substitution works

U-Substitution: The Most Important Method

Spot the clue: Part of the integrand is the derivative of another part. If you see $f(g(x))\cdot g'(x)$, let $u=g(x)$.

$$\int f\!\big(g(x)\big)\cdot g'(x)\,dx \;\xrightarrow{u\,=\,g(x)}\; \int f(u)\,du$$

Steps: (1) Pick $u$ = inner function   (2) Find $du = g'(x)\,dx$   (3) Replace everything in terms of $u$   (4) Integrate   (5) Substitute back to $x$

Ex: $\displaystyle\int x(x^2+1)^5\,dx$

$u=x^2+1 \Rightarrow du=2x\,dx \Rightarrow x\,dx=\tfrac{du}{2}$

$$= \frac{1}{2}\int u^5\,du = \frac{u^6}{12}+C = \frac{(x^2+1)^6}{12}+C$$

Ex: $\displaystyle\int\frac{1}{3x+1}\,dx$

$u=3x+1 \Rightarrow du=3\,dx \Rightarrow dx=\tfrac{du}{3}$

$$= \frac{1}{3}\int\frac{du}{u} = \frac{1}{3}\ln|u|+C = \frac{1}{3}\ln|3x+1|+C$$

U-Sub Fails When...

The derivative of the inner function is nowhere in the integrand. If $g'(x)$ (or a constant multiple of it) is missing, u-sub won't work — try integration by parts instead.

$$\int x\sin x\,dx$$ $\frac{d}{dx}[\sin x]=\cos x \neq x$. Neither is the derivative of the other. Use IBP.
$$\int x\cos(x^2)\,dx$$ $\frac{d}{dx}[x^2]=2x$ — and $x$ is right there! Let $u=x^2$.
