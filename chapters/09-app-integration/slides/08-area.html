9.8   Area by Integration

The Core Principle

Area = sum of infinitely thin strips. Choose vertical strips ($dx$) or horizontal strips ($dy$) based on which is simpler.

Vertical Strips ($dx$)

$$A = \int_a^b y\,dx = \int_a^b f(x)\,dx$$

Use when: curve given as $y=f(x)$, boundaries are vertical lines $x=a, x=b$.

Horizontal Strips ($dy$)

$$A = \int_c^d x\,dy = \int_c^d g(y)\,dy$$

Use when: curve given as $x=g(y)$, or boundaries are horizontal lines $y=c, y=d$.

Decision: Vertical vs Horizontal Strips?

  • If each vertical line crosses the region boundary exactly twice → use vertical strips
  • If each horizontal line crosses the region boundary exactly twice → use horizontal strips
  • If vertical strips would require splitting the region, try horizontal (and vice versa)

Watch for Sign Changes!

If $f(x)$ changes sign in $[a,b]$, the definite integral gives signed area (positive above, negative below $x$-axis). For geometric area, split at zeros and take absolute values:

$$\text{Geometric area} = \left|\int_a^c f(x)\,dx\right| + \left|\int_c^b f(x)\,dx\right|$$
$$\int_0^{2\pi}\sin x\,dx = 0 \neq \text{area}$$ Integral = 0, but geometric area = $2+2 = 4$!

Area Between Two Curves

$$\boxed{A = \int_a^b [f(x)-g(x)]\,dx \quad\text{where } f(x)\geq g(x) \text{ on } [a,b]}$$

Think: $A = \int(\text{top curve} - \text{bottom curve})\,dx$ or $\int(\text{right curve} - \text{left curve})\,dy$

Ex: Parabola $y^2=4x$ vs line $y^2=4y$

$y_U = 2\sqrt{x}$, $y_L = \frac{x^2}{4}$, intersection at $(0,0)$ and $(4,4)$.

$$A = \int_0^4\left(2\sqrt{x}-\frac{x^2}{4}\right)dx = \frac{16}{3}$$

Ex: Ellipse $\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$

By symmetry: $A = 4\int_0^a \frac{b}{a}\sqrt{a^2-x^2}\,dx$

$$= 4\cdot\frac{b}{a}\cdot\frac{\pi a^2}{4} = \boxed{\pi ab}$$

Quick Area Formulas Worth Memorizing

  • Parabola $y^2=4ax$ with latus rectum ($x=a$): $A = \dfrac{8a^2}{3}$
  • Ellipse: $A = \pi ab$
  • Circle: $A = \pi a^2$ (set $b=a$)
