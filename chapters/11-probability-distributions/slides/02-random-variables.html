11.1   Random Variables & Types

Core Concept

A random variable $X$ is a function from the sample space $S$ into $\mathbb{R}$. Each outcome $\omega \in S$ maps to a real number $X(\omega)$.

$$X : S \to \mathbb{R}$$

The range set: $R_X = \{X(\omega) \mid \omega \in S\}$

Discrete RV

Range of $X$ is countable (finite or countably infinite).

  • Used for counting quantities
  • “Number of heads”, “number of defectives”
$$X = 0, 1, 2, \ldots, n$$

Continuous RV

$X$ takes any value in an interval $I \subseteq \mathbb{R}$, with $P(X = x) = 0$ for every single point.

  • Used for measuring quantities
  • “Lifetime”, “length”, “distance”
$$X \in [a, b] \text{ or } [0, \infty)$$

Spot the Clue

  • “Find the values of the random variable” → define $X$ and list its range
  • “Number of ...” → discrete (counting)
  • “Lifetime”, “length”, “distance”, “amount” → continuous (measuring)
  • Coin tosses, dice rolls, ball draws → assign a number to each outcome

Ex: Two Coins Tossed — Number of Tails

Step 1: $S = \{TT, TH, HT, HH\}$

Step 2: $X(TT) = 2$, $X(TH) = 1$, $X(HT) = 1$, $X(HH) = 0$

Step 3: $X$ takes values $0, 1, 2$.

$$\boxed{\text{Values: } 0, 1, 2 \quad \text{Inverse image sizes: } 1, 2, 1}$$

Watch Out

$$S = \{HH, HT, TT\}$$ Forgot $TH$! Two coins have $2^2 = 4$ outcomes, not 3.
$$S = \{HH, HT, TH, TT\}$$ Always enumerate the FULL sample space first.

Trap: A discrete RV CAN be defined on a continuous sample space. E.g., $S = [0,1]$ with $X(\omega) = 10$ for all $\omega$ is discrete.
