11.4   Continuous Distributions — PDF vs CDF

Core Concept & Master Formulas

PDF $f(x)$: a density (NOT a probability).   CDF $F(x)$: accumulated probability.

$$\boxed{F(x) = \int_{-\infty}^{x} f(t)\,dt \quad \text{and} \quad P(a \le X \le b) = F(b) - F(a) = \int_a^b f(x)\,dx}$$
$$\boxed{\begin{aligned} &F(x) = \int_{-\infty}^{x} f(t)\,dt \\[4pt] &P(a \le X \le b) = F(b) - F(a) \end{aligned}}$$
$$\boxed{f(x) = \frac{dF(x)}{dx} = F'(x)}$$

For continuous RV, equality signs don’t matter: $P(a < X < b) = P(a \le X \le b)$.

When to Use PDF vs CDF

GoalUse PDF (integrate)Use CDF (directly)
Find $P(a \le X \le b)$$\int_a^b f(x)\,dx$$F(b) - F(a)$
Find $F(x)$ from $f(x)$$F(x) = \int_{-\infty}^{x} f(u)\,du$—
Find $f(x)$ from $F(x)$—$f(x) = F'(x)$
Find $P(X > a)$$\int_a^{\infty} f(x)\,dx$$1 - F(a)$

Spot the Clue

  • “Given $f(x)$, find the distribution function” → INTEGRATE piece by piece
  • “Given $F(x)$, find the density function” → DIFFERENTIATE
  • If $F(x)$ is already known, use $F(b) - F(a)$ — it’s FASTER than integrating
  • If you need MULTIPLE probabilities, build $F(x)$ first, then reuse it

Ex: Find $C$ and $P(1.5 < X < 3.5)$

Given $f(x) = Cx^2$ for $1 < x < 4$.

$$\int_1^4 Cx^2\,dx = C \cdot \frac{63}{3} = 21C = 1$$
$$C = \frac{1}{21}$$
$$P = \frac{1}{21}\!\int_{1.5}^{3.5}\!x^2\,dx = \frac{79}{126}$$

Ex: CDF to PDF

Given $F(x) = 0$ for $x < 0$; $x$ for $0 \le x < 1$; $1$ for $x \ge 1$.

$$f(x) = F'(x) = 1 \text{ for } 0 \le x < 1$$
$$P(0.2 \le X \le 0.7) = F(0.7) - F(0.2) = 0.5$$

Watch Out

$$f(a) = P(X = a)$$ $f(a)$ is a DENSITY, not a probability! For continuous RV, $P(X = a) = 0$ always.
$$P(X = a) = 0 \text{ always}$$ Only $P(a \le X \le b) = \int_a^b f(x)\,dx$ gives a probability.

Trap: When $f(x)$ is piecewise, don’t forget to split the integral at the breakpoints. Also, when building $F(x)$, each piece starts where the previous one ended.
