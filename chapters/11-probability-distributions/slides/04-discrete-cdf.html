11.3   Cumulative Distribution Function (Discrete)

Core Concept & Master Formula

The CDF accumulates probability from left to right. For discrete RV with values $x_1 < x_2 < \cdots$:

$$\boxed{F(x) = P(X \le x) = \sum_{x_i \le x} P(X = x_i)}$$

To recover the PMF from the CDF, take differences at jump points:

$$\boxed{f(x_i) = F(x_i) - F(x_{i-1})}$$

Properties

$$0 \le F(x) \le 1$$
$$F(-\infty) = 0$$
$$F(+\infty) = 1$$
$$P(x_1 < X \le x_2) = F(x_2) - F(x_1) \qquad P(X > x) = 1 - F(x)$$

CDF is a staircase function — each step height equals the pmf value at that point.

Spot the Clue

  • “Find the CDF” → accumulate pmf values left to right
  • Piecewise $F(x)$ given, “find pmf” → take differences at jump points
  • CDF is a staircase for discrete RV; each step = pmf value

Ex: PMF from a Piecewise CDF

$F(x) = 0$ for $x < -2$;  $0.25$ for $-2 \le x < -1$;  $0.60$ for $-1 \le x < 0$;  $0.90$ for $0 \le x < 1$;  $1$ for $x \ge 1$.

Jump points: $x = -2, -1, 0, 1$

$$f(-2) = 0.25 - 0 = 0.25$$
$$f(-1) = 0.60 - 0.25 = 0.35$$
$$f(0) = 0.90 - 0.60 = 0.30$$
$$f(1) = 1.00 - 0.90 = 0.10$$
$$\boxed{f(-2) = 0.25, \; f(-1) = 0.35, \; f(0) = 0.30, \; f(1) = 0.10}$$

Watch Out — Equality Signs Matter!

$$P(X \ge 2) = F(2)$$ Wrong! $F(2) = P(X \le 2)$, not $P(X \ge 2)$.
$$P(X \ge 2) = 1 - F(1)$$ $P(X \ge 2) = 1 - P(X \le 1) = 1 - F(1)$
$$P(X < 3) = F(3)$$ For discrete: $P(X \le 3) \neq P(X < 3)$ since $P(X=3) \neq 0$.
$$P(X < 3) = F(3) - f(3)$$ Subtract the jump at $x = 3$ to exclude it.

Trap: For continuous RV, $P(X < a) = P(X \le a)$. For discrete RV, they are DIFFERENT.
