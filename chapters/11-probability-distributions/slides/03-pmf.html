11.2   Probability Mass Function (PMF)

Core Concept

For a discrete RV $X$ with values $x_1, x_2, \ldots$, the probability mass function assigns a probability to each value:

$$f(x_k) = P(X = x_k), \quad k = 1, 2, 3, \ldots$$
$$\boxed{f(x_k) \ge 0 \text{ for all } k, \quad \text{and} \quad \sum_k f(x_k) = 1}$$

Spot the Clue

  • “Find the probability mass function” or a table with $x$ and $f(x)$ values
  • Given $f(x) = ck$ or $f(x) = c^2$ type expressions with unknown constant → set $\sum f(x) = 1$ and solve for the constant
  • “Find the value of $k$” in a pmf table → immediately use $\sum f(x) = 1$

Ex: Find $k$ and $P(2 < X < 6)$

Given: $f(1) = k$, $f(2) = 2k$, $f(3) = 6k$, $f(4) = 5k$, $f(5) = 6k$, $f(6) = 10k$.

Step 1: $\sum f(x) = k + 2k + 6k + 5k + 6k + 10k = 30k = 1$

$$k = \frac{1}{30}$$

Step 2: $P(2 < X < 6) = f(3) + f(4) + f(5) = \frac{6 + 5 + 6}{30}$

$$\boxed{k = \frac{1}{30}, \quad P(2 < X < 6) = \frac{17}{30}}$$

Watch Out

$$\sum f(x) = 1 \implies c = \tfrac{1}{5} \text{ or } c = -\tfrac{1}{2}$$ Accepting $c = -\frac{1}{2}$. But $f(x) \ge 0$ required — reject negative!
$$c = \tfrac{1}{5} \;\checkmark \quad c = -\tfrac{1}{2} \;\times$$ Always check BOTH: non-negativity AND total = 1.

Trap: $P(2 < X < 6) = f(3) + f(4) + f(5)$, NOT including endpoints 2 and 6. But $P(2 \le X < 5)$ includes $f(2)$.
